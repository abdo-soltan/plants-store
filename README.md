# Verdant — Ornamental Plants E-Commerce

Premium plant boutique storefront built with React 18, Vite, Tailwind CSS, React Router, Framer Motion and Lucide icons.

## Getting started

```bash
npm install
npm run dev
```

Then open the printed local URL (usually http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Notes

- Cart and Wishlist persist to `localStorage`.
- Dark mode toggle persists to `localStorage` and respects the class strategy (`darkMode: 'class'` in Tailwind).
- Product images use Unsplash source placeholders — swap with real product photography in `src/data/products.js`.
